Java Web Start

Si vous avez le Java Web Start, vous pouvez lancer le logiciel de démonstration intelligent des préférences du Web pour tous en cliquant le lien suivant:

Logiciel de démonstration intelligent pour les préférences du Web pour tous

Installateur pour Windows

Pour installer le logiciel de démonstration intelligent pour les préférences du Web pour tous sur votre ordinateur, veuillez faire un clic droit sur le lien suivant et choisir "enregistrer sous" à partir du menu secondaire:

Web4AllPreferenceWizardDemo.exe

Pour visioner les instructions d'installation, veuillez cliquer le lien suivant:

instructions d'installation

(Note: L'accès au logiciel de démonstration avec des lecteurs d'écran exige le pont d'accès de Java pour Windows. Ce pont est disponible au site Web suivant:

Pont d'accès de Java pour Windows
