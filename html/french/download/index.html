Version publique du Web pour tous

Le Web pour tous lancé en tant que logiciel d’exploitation libre

Le 31 mars 2006

Le centre ATRC de l’Université de Toronto est heureux d’annoncer que Le Web pour tous a été lancé en tant que logiciel d’exploitation libre téléchargeable sans frais de ce site Web. Cette version fournit aux administrateurs de systèmes d’exploitation à utilisateurs multiples la technologie de base et les plugiciels du Web pour tous. S’ils veulent monter un système entièrement fonctionnel dont peuvent bénéficier les utilisateurs d’Internet handicapés, les administrateurs de systèmes peuvent allier Le Web pour tous aux logiciels de technologies d’aide (TA) de sociétés indépendantes et aux plugiciels appropriés de Web pour tous pour configurer toutes ces TA. On trouve ici nombre de plugiciels de Web pour tous pour les TA sélectionnées. Les administrateurs de système peuvent obtenir les TA auprès des vendeurs pertinents.

Les développeurs de technologies d’aide sont encouragés à intégrer leurs produits à Web pour tous en implémentant les plugiciels du Web pour tous compatibles avec l’interface applicative d’exploitation libre (API) de Web pour tous pour les plugiciels. Un manuel du programmeur et le code source pour les plugiciels toutes caractéristiques incluses de Web pour tous et bon nombre de plugiciels prototypes sont offerts pour soutenir cette démarche.

Le Web pour tous a été écrit en Java et conçu en vue de s’exécuter sur n’importe quel système d’exploitation pris en charge par le moteur d’exécution Java, comme Windows, Macintosh et Linux.

Cette version est disponible comme code source.

Version de Web pour tous pour Windows XP

Le Web pour tous est offert pour Windows XP sous format exécutable et autoinstallable. Il y a des installateurs distincts en français et en anglais prêts à télécharger. Le client et les plugiciels de Web pour tous sont inclus pour bon nombre de technologies d’aide de sociétés indépendantes. 

Remarques sur la version :

Prise en charge des navigateurs de fichiers ou de carte à puce
Lorsque vous installez Le Web pour tous, vous pouvez sélectionner l’une des options suivantes :
Option n° 1 : Prise en charge du lecteur de carte à puce USB
Des lecteurs de carte à puce USB peuvent être achetés pour Windows XP. L’ATRC a testé ce type de lecteur de carte à puce et a déterminé qu’il est compatible avec Le Web pour tous sous Windows XP. Ce lecteur de carte à puce est le SCR335 Chipdrive de SCM Microsystems que vous trouverez chez de nombreux détaillants. Vous pouvez en consulter les spécifications sur le site Web de SCM à l’adresse suivante : http://www.scmmicro.com/security/SCR335.html 
Prenez note que le lecteur de carte Hitachi pris en charge par l’ancienne version Windows 2000 du Web pour tous n’est pas pris en charge par cette version parce que l’assistance technique pour le système n’est pas fournie pour ce lecteur de carte pour Windows XP par Hitachi, le détaillant de lecteur de carte à puce.
Option n° 2 : Assistance technique pour le navigateur
Le Web pour tous pour Windows XP prend en charge l’enregistrement et la récupération des préférences sur tout dispositif de lecture/écriture pris en charge par le système de fichiers Windows XP, par exemple, un Memory Stick USB. C’est-à-dire que les administrateurs ont l’option de permettre aux utilisateurs d’enregistrer leurs préférences sur ces dispositifs au lieu des cartes à puce. Prenez note que le navigateur de fichiers prend en charge les appareils autoparlants (par synthétiseur à partir du texte) et les très grands caractères.

Services d’intégration et de personnalisation de Web pour tous

Nous sommes disponibles à des conditions fixées d’avance pour aider à l’installation de Web pour tous dans les bibliothèques, les salons Internet, les universités ou autres. Nous pouvons assurer des services de développement de logiciels en vue de personnaliser Le Web pour tous afin qu’il réponde à vos besoins spécifiques et développer les plugiciels de Web pour tous destinés aux technologies d’aide de sociétés indépendantes. Veuillez communiquer avec nous, web4all@atrc.utoronto.ca, pour plus de renseignements.
