Le Web pour tous pour Windows XP

L’ATRC met cette version du logiciel Le Web pour tous à la disposition de tous les sites du Programme d’accès communautaire (PAC).

La nouvelle version du logiciel sera distribuée au moyen de fichiers binaires distincts comprenant la version anglaise et la version française du client et des plugiciels du Web pour tous. Pour migrer de Windows 2000 à Windows XP, les sites du PAC pourront installer les technologies d’assistance séparément à partir du CD de Windows 2000, à l’exception du clavier virtuel dynamique (CVD) de l’ATRC, pour lequel un installateur téléchargeable sera offert. Signalons que les technologies d’assistance qui ont été fournies aux sites du PAC pour la version Windows 2000 ne feront pas partie de la trousse d’installation téléchargeable du Web pour tous, car l’ATRC ne possède pas de licence qui lui permettrait de distribuer ces fichiers.

Par ailleurs, le logiciel Le Web pour tous pour Windows XP ne prendra pas en charge le lecteur de carte à puce qui est actuellement pris en charge par la version pour Windows 2000, car Hitachi, le fournisseur du lecteur de carte ne prend pas en charge Windows XP. Deux options sont toutefois proposées pour assurer le fonctionnement du Web pour tous sous Windows XP. Au moment de l’installation du Web pour tous, on vous proposera les deux options suivantes :

Option n° 1 : Lecteur de carte à puce USB
Des lecteurs de carte à puce USB pour Windows XP peuvent être achetés pour Windows XP. L’ATRC a testé un tel lecteur et a confirmé qu’il fonctionnait avec Le Web pour tous sous Windows XP. Ce lecteur de carte peut être employé avec les cartes à puce qui ont été distribuées aux sites du PAC avec la version du Web pour tous pour Windows 2000. Il s’agit du lecteur de carte à puce SCR335 Chipdrive de SCM Microsystems, que vous pouvez acheter d’un certain nombre de détaillants. On pourra consulter les spécifications de ce lecteur sur le site Web de SCM, à l’adresse suivante : http://www.scmmicro.com/security/SCR335.html 
Nous soulignons que l’ATRC ne pourra pas fournir de soutien en cas de problème concernant ce lecteur de carte USB ou tout autre lecteur de carte. Les essais réalisés n’ont toutefois permis de déceler aucun problème.
Option n° 2 : Prise en charge du navigateur de fichiers
Le Web pour tous pour Windows XP permet d’enregistrer et de récupérer les préférences sur n’importe quel dispositif de lecture-écriture pris en charge par le système de fichiers de Windows XP, comme les barrettes de mémoire USB. Cela signifie que les sites du PAC peuvent permettre aux utilisateurs d’enregistrer leurs préférences sur ce type de dispositif au lieu des cartes à puce.

L’ATRC offrira du soutien technique jusqu’au 31 mars 2006, afin d’aider les administrateurs à installer et à mettre à niveau le logiciel. Pour obtenir de l’assistance, veuillez communiquer avec l’ATRC à l’adresse web4all@atrc.utoronto.ca.

Installateur du Web pour tous pour Windows XP

Étapes à suivre pour télécharger et installer Le Web pour tous pour Windows XP :

  1. Téléchargez l’installateur sur la machine où vous voulez installer Le Web pour tous.
  2. Faites le double clic sur l’icône de l’installateur pour le lancer. Utilisez les répertoires d’installation par défaut suggérés par les boîtes de dialogue de l’installateur.
  3. Après l’installation, un raccourci du Web pour tous se placera sur le bureau. Faites le double clic sur l’icône du Web pour tous pour le lancer.
  4. Le Web pour tous s’ajoute également au menu des « Programmes », et aux éléments initiaux. Le Web pour tous se lance automatiquement chaque fois que le poste de travail est redémarré.

Vous pouvez assigner l’enfoncement d’une touche au lancement du Web pour tous à partir du clavier :

  1. Cliquez à droite sur l’icône du raccourci du Web pour tous sur le bureau, et choisissez « Propriétés » du menu contextuel.
  2. Dans la boîte de dialogue subséquente « Propriétés de Le Web pour tous », sélectionnez l’onglet « Raccourci » pour accéder aux propriétés de la touche de raccourci.
  3. Saisissez l’enfoncement de touche voulu dans la zone du texte de la « Touche de raccourci ».
  4. Cliquez sur « Appliquer », puis sur « OK ».

Téléchargez l’installateur du Web pour tous : Le-Web-pour-tous-PAC.exe.

Sélecteur de préférences autoparlant

La version du Web pour tous pour Windows XP inclut un dialogue de sélecteur de fichier autoparlant. Pour qu’il fonctionne correctement, un synthétiseur de parole doit être fourni avec votre système Windows. Windows XP, en français commen en anglais, comporte un synthétiseur de parole préinstallé. Afin que le français soit prononcé correctement, vous devez télécharger et installer un synthétiseur de parole français :

  1. Téléchargez et installez le lecteur Microsoft Reader de cette page Web :
    http://www.microsoft.com/reader/downloads/pc.asp
  2. Téléchargez et installez le composant synthèse vocale Microsoft Reader en français de cette page Web :
    http://www.microsoft.com/reader/fr/downloads/tts.asp

Installateur du clavier virtuel dynamique (CVD)

  1. Téléchargez l’installateur dans la machine où vous voulez installer le CVD.
  2. Faites le double clic sur l’icône de l’installateur pour le lancer. Utilisez les répertoires d’installation par défaut suggérés par les boîtes de dialogue de l’installateur.

Tééchargez l’installateur du CVD : CVDFrançais.exe.
