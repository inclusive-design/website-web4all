MouseKeysSetup.exe

MouseKeysSetup

MouseKeysSetup — configure Windows MouseKeys accessibility features.

Synopsis

MouseKeysSetup speed acceleration

Description

This utility can be used to configure the Windows mouse keys accessibility parameters. MouseKeys allows the user to use the numeric keypad on the keyboard to control the mouse movements.

Invoked with no arguments, all settings are set to the system default. If any arguments are provided, all arguments must be provided. To maintain the system default, use the relevant default value (described below).

Parameters

speed
The maximum speed of the mouse movement when an arrow key is held down. Valid range is between 10 and 360.
Default: 60
acceleration
The acceleration of the mouse movement, expressed as the time (in milliseconds) to reach the maximum speed. Valid range is between 1000 and 5000.
Default: 5000

Return Value

The utility prints progress information and any error messages to the console output.

Upon successful completion, the program exits with an exit status of 0.

Any error condition will result in an exit status of -1.
